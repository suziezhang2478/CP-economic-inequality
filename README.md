# CP-economic-inequality
R code for comparative politics writing sample on economic inequality and the social base of authoritarian support in democracies


# Data availability
Data sources:
1. World Values Survey (WVS)
2. World Development Indicators (WDI)
3. Standardized World Income Inequality Database (SWIID)
4. World Inequality Database (WID)
5. Varieties of Democracy (V-Dem) Regimes of the World index

Due to redistribution restrictions, WVS data must be downloaded from the WVS website. WDI data from the World Bank can be accessed directly from the R command in the script. SWIID, WID, and V-Dem regime score files are in the data folder.
